Have you ever been playing a game and thought... 'Man this game is so much like another game that I've played'?

Gamebonder is a simple web application that allows you to bond two games together and provide a description as to why you believe they are alike. Simple! We keep track of the bonds that you have created and allow you some unique views of how games are related.

This is a work in progress so please come back often to see the updates!
